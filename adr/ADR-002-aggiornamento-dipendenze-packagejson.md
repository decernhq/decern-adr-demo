# Aggiornamento dipendenze package.json

**Status:** Proposed

**Author:** Alessandro Palazzesi

## Context

È emersa la necessità di mantenere il progetto aggiornato per garantire sicurezza e compatibilità con le ultime versioni delle librerie utilizzate.

## Options Considered

- Aggiornare solo le dipendenze critiche
- Non aggiornare le dipendenze e mantenere la versione attuale

## Decision

Aggiornare il package json

## Consequences

Conseguenze positive includono miglioramenti in termini di sicurezza e nuove funzionalità. Conseguenze negative potrebbero includere incompatibilità con il codice esistente e la necessità di test approfonditi.
